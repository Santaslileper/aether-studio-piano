How to expand the note array

Each entry in var d follows this format:

{type: "note",  note: "C#5",  time: 1.316,  duration: 0.188}

note — pitch + octave. Sharps use # (e.g. "D#5", "G#2").
time — seconds from the start of the piece. One sixteenth note ≈ 0.188s at this tempo.
duration — length in seconds. Used loosely (VexFlow derives duration from slot spacing).

Common durations at ~80 BPM (sixteenth = 0.188s, 3/8 time):

Sixteenth  = 0.188   →  slot step ×1
Eighth     = 0.376   →  slot step ×2
Quarter    = 0.752   →  slot step ×4
D. Quarter = 1.128   →  fills one full 3/8 measure

Notes within 0.05s of each other are treated as a chord. Octave 4+ renders in the treble staff; octave 3 and below in the bass staff.
